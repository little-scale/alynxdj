# ALYNXDJ HELP screen text -> build/help.h + build/help.bin.
# One source line is one screen row. The build validates:
#   - UPPERCASE display text using glyphs present in the Lynx font
#   - no more than 38 characters per line
#   - no more than 16 rows per page
# A blank line is a blank row. "---" starts a page. "@VERSION" bakes the
# current version and build stamp. A line containing ":" is drawn inverted.

NAVIGATION:
DPAD             MOVE IN SCREEN
HOLD A + DPAD    MOVE ON SCREEN MAP
HOLD A TAP B     PLAY / STOP CONTEXT
OPTION 1 TAP     PLAY ALL FROM CURSOR
OPTION 1 + L/R   SELECT TRACK
OPTION 1 + B     MUTE TRACK
OPTION 1 + A     SOLO / UNSOLO TRACK
OPTION 2 ON SONG LIVE MODE

HELP:
DPAD             TURN PAGE
FROM TABLE HOLD A + UP TO ENTER
HOLD A + DOWN    BACK TO TABLE
---
CELL EDITING:
TAP B            INSERT / AUDITION
HOLD B + L/R     EDIT SMALL STEP
HOLD B + U/D     EDIT BIG STEP
HOLD B TAP A     CUT FIELD + COPY
TAP B,B          PASTE FIELD

BLOCK EDITING:
HOLD B + HOLD A  SELECT BLOCK
DPAD             EXTEND BLOCK
TAP B            COPY BLOCK
HOLD B TAP A     CUT BLOCK + COPY
TAP A            CANCEL BLOCK
TAP B,B          PASTE BLOCK
---
MINT AND CLONE:
B,B ON EMPTY SONG / CHAIN CELL
= MINT NEXT FREE CHAIN / PHRASE
B,B ON USED SONG / CHAIN CELL
= SLIM CLONE CHAIN / PHRASE

STRUCTURE:
PHRASE = 16 NOTE ROWS
CHAIN  = 16 PHRASE REFS + TRANSPOSE
SONG   = 128 ROWS X 4 TRACKS
NOTES, INSTRUMENTS + TABLES ARE
GLOBAL POOLS, NOT OWNED BY A TRACK
---
INSTRUMENTS: SELECT INSTR 00-1F
TYPE       TONE / NOISE / WAV / KIT
VOL        ENVELOPE PEAK 00-7F
ATK        ATTACK TIME 0-F
HOLD       PEAK TIME, F = INDEFINITE
DCY        DECAY TIME, 0 = SUSTAIN
TSP        SIGNED SEMITONE TRANSPOSE
TABLE      MACRO TABLE, -- = NONE
TBS        0 PER NOTE, 1-F TICK RATE

TONE / NOISE:
SWP        SIGNED PITCH SWEEP
VIB        SINE PITCH LFO
TRM        DOWN-RAMP VOLUME LFO
TAPS/SEED  LYNX LFSR TIMBRE + STATE
TAP B WHILE STOPPED TO AUDITION
---
WAV:
WAVE 0-7 LOOPS A 32-BYTE USER WAVE
WAVE -- USES THE HARDWARE TRIANGLE
WAVE SCREEN EDITS ALL 8 USER WAVES

KIT:
KIT 0-7 SELECTS A SAMPLE BANK
NOTE SEMITONE SELECTS ITS SAMPLE
EVERY TRACK MAY PLAY WAV OR KIT
MAX 2 TIMER-FED DAC VOICES AT ONCE
THIRD DAC VOICE STEALS THE OLDEST
TABLES:
16 ROWS OF VOL / TSP / COMMAND
TBS SETS NOTE OR TICK PLAYBACK RATE
H LOOPS; OTHERWISE LAST ROW HOLDS
ACTIVE ROW FOLLOWS TOP-BAR TRACK
---
COMMANDS A-L:
A XX  RUN TABLE XX ON THIS NOTE
B XX  ADD SIGNED TAPS, 00 RESET
C XY  CHORD 0,+X,+Y
D XX  DELAY NOTE XX TICKS
E XY  ENVELOPE ATK X / DCY Y
F XX  SIGNED FINE PITCH
G XX  SIGNED TAP GLIDE + PERIOD
H XX  HOP TO ROW XX BEFORE PLAYING
I XX  ITERATION PLAY MASK
J XY  PASS-GATED TRANSPOSE
K XX  KILL AFTER XX TICKS
L XX  SLIDE AT XX/16 SEMI PER TICK
---
COMMANDS N-Z:
N XX  LIVE LFSR TAPS OVERRIDE
O XY  LYNX II LEFT/RIGHT ATTENUATION
P XX  SIGNED PITCH BEND
R XY  RETRIGGER PERIOD + DECAY
S XX  WAV/KIT TIMER RATE
T XX  GLOBAL TEMPO, FLATTENS GROOVE
V XY  SINE VIBRATO SPEED + DEPTH
W XX  SHORTEN ROW TO XX TICKS
X XX  SET NOTE ENVELOPE PEAK
Z XX  RANDOM NOTE GATE

COMMANDS STEP A-Z ALPHABETICALLY
---
PLAYBACK:
SONG LOOPS ITS CONTIGUOUS GROUP
LIVE QUEUES CHAIN STARTS / STOPS
LIVE CHANGES LAND ON A 16-ROW BAR

SYNC:
IN       ONE COMLYNX PULSE PER ROW
IN24     USB MIDI CLOCK VIA PICO
MIDI     CHANNELS 1-4 PLAY INST 1-4
IN / IN24 SHOW WAIT UNTIL FIRST PULSE
THE HELP SCREEN STOPS TRANSPORT
---
FILES:
SAVE      PACK SONG TO 2 KB EEPROM
LOAD      RESTORE THE SAVED SONG
NEW       CLEAR TO DEFAULTS
DEMO      LOAD THE FACTORY SONG
PURGE     REMOVE UNUSED OBJECTS
A SONG TOO LARGE TO SAVE IS REFUSED

LIMITS:
4 TRACKS       32 INSTRUMENTS
64 PHRASES     32 CHAINS
128 SONG ROWS  16 TABLES
8 USER WAVES   64 SAMPLE SLOTS
ALYNXDJ:
@VERSION
